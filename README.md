声の大きさ信号機 (Voice Traffic Light)

教室の環境音をマイクで拾い、騒音レベルを信号機（青・黄・赤）の色とメーターでリアルタイムに可視化するWebアプリケーションです。

「静かにしなさい」と口頭で注意する代わりに、画面を見せるだけで児童生徒自身が「今の声の大きさ」に気づき、自発的に行動を調整できるように設計されています。

🎯 主な特徴

リアルタイム可視化: 音の大きさに応じて、信号機の色が変化し、キャラクター（フクロウなど）が状況を伝えます。

3つのプリセットモード: 活動内容に合わせて、感度としきい値をワンタップで切り替えられます。

📖 自習モード: 基準が厳しく、少しの私語でも反応します。

🏫 授業モード: 標準的な設定です。

🗣️ グループ活動: 話し合い活動用に、ある程度の賑やかさを許容します。

カスタマイズ機能: 教室の広さやマイクの性能に合わせて、感度やしきい値を自由に調整できる詳細設定メニューを搭載。

優しい警告: しきい値を超えた時は、不快なブザー音ではなく、柔らかい電子音や動物の鳴き声風の音で通知します。

プライバシー保護: 音声データはブラウザ内でのみ解析され、サーバーへの送信や録音は一切行われません。

🚀 使い方

起動: voice_traffic_light.html をWebブラウザで開きます。

測定開始: 画面中央の「測定スタート」ボタンを押します。（初回のみマイクの使用許可を求められます。「許可」を選択してください）

モード選択: 活動内容に合わせて、右側のパネルから「自習」「授業」「グループ活動」のいずれかを選択します。

詳細調整: 必要に応じて、右上のメニューボタン（≡）から、マイク感度や色の変わる基準値をスライダーで微調整できます。

🛠️ 技術仕様

言語: HTML5, CSS3, JavaScript (ES6+)

フレームワーク: なし (Vanilla JS)

スタイル: Tailwind CSS (CDN読み込み), FontAwesome (アイコン)

音声処理: Web Audio API

ScriptProcessorNode ではなく、軽量な AnalyserNode を使用して音圧レベル（RMS）を算出しています。

⚠️ 注意事項

マイク必須: このアプリを使用するには、マイク付きの端末（PC、タブレット、スマホ）が必要です。

ブラウザ対応: Google Chrome, Edge, Safari, Firefoxなどのモダンブラウザで動作します。

ハウリング: スピーカーの音をマイクが拾わないよう、音量は控えめにするか、端末の配置を調整してください。
